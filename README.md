### **ARK Survival Ascended QA Bug Reporting Project**



###### **Project Overview:**

This repository showcases my QA testing and bug reporting work for ARK Survival Ascended.

The project focuses on identifying, documenting, and tracking in game issues using structured bug reports.



###### **The goal of this project is to demonstrate:**

* Real world game testing skills
* Clear and structured bug documentation
* Understanding of QA processes and workflows



###### **Tools \& Technologies Used:**

* Kanban Board – Bug tracking and workflow management
* Microsoft Excel – Test case design and documentation
* GitHub – Project hosting and version control
* GitHub Bug Tracking Board



###### **You can view all reported bugs and their current status here:**

[ARK\_Bug\_Reports](https://github.com/users/nishith-shettyy/projects/3)



###### **Scope of Testing:**

The testing covered multiple gameplay systems, including:

* Creature behavior \& physics
* Environmental interactions
* Character customization issues
* Object spawning \& collision problems
* Animation and ragdoll glitches



###### **Sample Bugs Reported:**

Some of the issues documented in this project include:

* Shoulder pets deforming based on character body proportions
* Water physics allowing creatures to partially exit water incorrectly
* Creatures appearing with broken/origami like textures
* Loot crates spawning in inaccessible locations
* Extreme ragdoll glitches on creature death



###### **Each bug report includes:**

* Summary
* Description
* Steps to Reproduce
* Expected vs Actual Result
* Severity \& Reproducibility
* Environment
* Video Link
